Human-facing public documentation lives here.

- `docs/YOUTUBE_DATA_PIPELINE.md`: YouTube channel, video, transcript, cue, micro-event, and timeline pipeline behavior.
- `docs/YOUTUBE_DATA_PIPELINE_TODO.md`: remaining pipeline backlog.

- `docs/HOME_PC_DEPLOYMENT.md`: short entrypoint that routes to the local native deployment guide.
- `docs/ARCHITECTURE_LINTING.md`: Import Linter clean architecture gate, layer rules, and violation repair guidance.
- `docs/ARCHIVE_PUBLISH.md`: archive publish API, object layout, cache policy, and required environment variables.
- `docs/CICD.md`: current GitHub Actions status. Workflows are manual checks only.
- `ops-ui/docs/FRONTEND_ARCHITECTURE.md`: Next.js Ops UI structure, BFF, state, and screen organization.
- `ops-ui/docs/API_CONTRACT.md`: OpenAPI export and generated frontend type update workflow.
