# Docs Index

사람이 프로젝트를 이해하기 위한 문서 모음이다. agent-facing 지침은 루트 `AGENTS.md`와 `vaults/`를 사용한다.

- `docs/PROJECT_OVERVIEW.md`: 이 프로젝트가 무엇이고, CLI/FastAPI 구조로 어떻게 동작하며, 어떻게 실행/확장하는지 설명한다. 프로젝트를 처음 보는 개발자나 사용자가 읽는다.
- `docs/AWS_DEPLOYMENT.md`: Terraform으로 AWS EC2에 `codex-demo` CLI를 배포하고, S3 bucket을 읽기 전용으로 마운트해 Codex가 탐색하게 하는 절차를 설명한다.
